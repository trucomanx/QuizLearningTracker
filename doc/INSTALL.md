# quiz-learning-tracker

A simple application to generate quizzes from structured data and track learning performance over time.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/quiz_learning_tracker/

```bash
pip install --upgrade quiz_learning_tracker
```

Using:

```bash
quiz-learning-tracker
```

## Install from source
Installing `quiz-learning-tracker` program

```bash
git clone https://github.com/trucomanx-desktop/QuizLearningTracker.git
cd QuizLearningTracker
pip install -r requirements.txt
cd src
python -m build
pip install dist/quiz_learning_tracker-*.tar.gz
```
Using:

```bash
quiz-learning-tracker
```

## Uninstall

```bash
pip uninstall quiz_learning_tracker
```
